Melbourne Bioinformatics Training Program 2026 – Application Repository

This repository contains my completed programming exercises, data visualizations, and scripts for the Melbourne Bioinformatics Training Program 2026 application.

Data Files

GSE60450_GeneLevel_Normalized(CPM.and.TMM)_data.csv – gene expression dataset used for exercises

Scripts / Plots

boxplots.R – creates boxplots comparing gene expression across samples

Usage

Clone the repository to your local computer:

git clone https://github.com/yourusername/BioInformatics-training-program-application-2026.git


Open the R scripts in RStudio.

Ensure the required packages are installed:

install.packages("tidyverse")


Run the scripts to reproduce analyses and visualizations.
